Features

  • Buddy Room — Create or join a 6-digit room. Share item status, notes, max bids live with up to 3 auction buddies. Rooms auto-expire after 24 hours.
  • Real-time Sync — Changes appear instantly on all connected phones via Supabase postgres_changes. Presence tracking shows who's online with green dots.
  • Shared Comps — When one buddy searches for comps, results are cached in Supabase. Other buddies skip the Apify call and use the shared result — saves ~$0.03/search.
  • Offline Queue — Edits made without signal are queued in localStorage. Automatically flushed when connection returns.
  • Activity Feed — Live feed in Settings showing who changed what on which item.
  • Sync Status Dot — Floating green pill in top-right of all screens. Shows room code, tapping opens Settings.
  • Search Radius — Configurable 50-300mi radius for comp searches.
  • Region Picker — 7 regions: SoCal, NorCal, Phoenix, Vegas, Dallas, Houston, Orlando. Multi-select, expands Craigslist + Facebook coverage.
  • Results per Source — 5-50 slider controls how many listings to pull from each source.
  • Apify Usage Gauge — Cost tracking with color-coded gauge bar and monthly budget slider.
  • NPA Code Stripping — Internal manufacturer codes (KRT800GJF) stripped from comp queries, preventing wasted Apify searches.
